# Threats: Vermus

Static HTML document for 'Threats: Vermus'.

Requirements:

  * [Node.js](http://nodejs.org/).

To set-up your environment execute:

    $ npm install

To build execute:

    $ npm run build

To test:

    $ npm run start

And open http://localhost:8000 in your browser.

To clean build:

    $ npm run clean

To publish as a PDF:

    $ npm run publish
